This is a collection of links to weather stations around Oslo, Norway.
The information comes from Holfuy.com. 
This page picks up widgets from holfuy.com and pressent it as a collection, with differnet  layout depending on use of desktop browser of phone.
It also ha a link to a good weather station at the bottom.
It also has link to nearby webcam, if there are any.

The links to th differnt sites is stored in an array: [num of weather station, name og station, link to webcam].

It uses a custom background for the widges, easy to make your own. 

A version of this site is per today 2021-05-15 up at http://jed-design.com/fly.htm

It is easy to adapt the page for your own needs. It is not so much commented, that may improve later.
